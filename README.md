# 马拉松赛事志愿者管理平台

全栈马拉松赛事志愿者管理平台，支持赛事方发布招募、志愿者报名、审核排班、签到打卡和赛后评价。

## 功能简介

- **赛事招募**：赛事方发布志愿者招募，设置岗位（补给站/指引/医疗/计时/物资发放），说明工作内容和时间段
- **志愿者报名**：志愿者选择感兴趣的岗位，填写可服务时间段和个人信息
- **审核与通知**：赛事方审核报名，通过后自动创建通知和排班记录
- **排班表**：志愿者查看自己的排班详情（点位、时间段、对接人联系方式）
- **签到打卡**：比赛当天签到和签退
- **赛后评价**：赛事方给志愿者评分并发放证书
- **管理后台**：全平台数据统计、志愿者名单导出 CSV

## 技术栈

- 前端：Vite + React 18 + TypeScript + React Router v6
- 后端：Express + TypeScript + better-sqlite3
- 认证：JWT + bcryptjs

## 运行方式

```bash
npm install
npm run dev
```

前端启动在 http://localhost:5173，后端启动在 http://localhost:3001。

首次启动会自动创建 SQLite 数据库，插入管理员、赛事方、志愿者账号以及一条示例赛事和 5 个岗位。
